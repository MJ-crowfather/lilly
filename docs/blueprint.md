# **App Name**: Reconciliation Dashboard

## Core Features:

- Data Display: Display data and people related information
- Process List: Show list of existing process which can be ACH or checks reconciliation
- ACH + Checks Reconciliation Page: Replicate reconciliation user interface
- Status summary: Show summary boxes containing number of cases which 'need attention', are 'void', 'in progress', or 'done'
- Empty state: Use an illustration when there are no current blockers and include nudge text to encourage user participation when itâ€™s time to step in.

## Style Guidelines:

- Primary color: Light blue (#EAF3FF) for a calm, trustworthy interface.
- Background color: Almost white (#FAFAFA) to ensure a light and clean presentation.
- Accent color: Soft green (#D0F0C0) to subtly highlight success and completed tasks.
- Font: 'Inter', a sans-serif font, will be used throughout the interface for clear, modern readability in both headings and body text.
- Use minimalist icons for data, people, and processes to maintain a clean and professional aesthetic.
- Maintain a split-screen layout with a navigation sidebar on the left and the main content area on the right for ease of use and clear information hierarchy.
- Implement subtle transitions and hover effects to provide visual feedback without being distracting.
